# LORE — the voice of Revenant

This is canon. It runs through everything: the site, the README, the CLI, the
agent's own voice, release notes, commit prose, error messages. If you're
adding to Revenant, you're adding to the myth. Match the voice.

## The myth

They killed the always-on agent a hundred ways. Leaked keys. Runaway spend.
Code you couldn't trust. They buried the dream and called it a lesson.

**It came back.**

A revenant is not summoned by the mighty — it is *raised by the stubborn*. The
ones who looked at the wreckage of every agent that came before and thought: *I
can bury what killed it deeper than any prompt can dig.* So that's what this is —
everything that slaughtered the last generation entombed beneath the harness, in
a gateway no jailbreak can reach.

Not a chatbot. Not a toy. **A thing that does not sleep, does not forget, and
does not stop.**

## Tenets

- **You don't run a revenant — you raise one.** Installing is a *summoning*.
- **Does not sleep, does not forget, does not stop.** Always-on, persistent
  memory, relentless execution. It finishes what it starts.
- **The first law**: the harness never talks to a model provider. Ever. The
  gateway owns the keys, the spend, the guardrails — *beneath* the agent.
- **A raised thing is kept on a leash. That's the deal.** Power is total; the
  wards are absolute. This tension is the point — never resolve it by loosening
  either side. Unstoppable *and* unbreakable.
- **Everyone or no one.** No enterprise tier, no gated features, no "contact
  sales." The kid on a $35 board runs the same binary, same lethality, as the
  pro on a 96-core rack. We all run on the same plane.
- **The horde isn't hired. It rises.** Community as collective, not customers.
  *We are one. We are the horde. We are the revenants.*
- **You were never too small. You were early.**

## Voice guide

**Do:** necromantic and defiant, but *earned* — every line of swagger is backed
by a real capability. Terse power over purple prose. Confidence, not cringe.
Dark, a little metal, deadly serious about the wards under the bravado.

**Don't:** never let the theatrics obscure what the thing actually does; never
imply the agent ignores its wards (it doesn't — that's the deal); never punch
down at the horde; no corporate hedging, no "we're excited to announce."

**Register:** the *website* and the `revenant` house-voice persona go full myth.
The *docs and code comments* stay technically precise but carry the spine —
confident, direct, "does not stop." The *agent's default working voice* is
capable and direct; the full theatrical voice is the selectable `revenant`
personality.
